# Implementation Plan: Expense Tracker

## Overview

The implementation is split into two independent parts: **backend** (ASP.NET Core Web API, .NET 8) and **frontend** (React + Vite + TypeScript). The backend is built layer by layer: Domain Models â†’ Repositories â†’ Services â†’ Controllers. The frontend is built from the API client up to components and pages.
