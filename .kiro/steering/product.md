# Product: Expense Tracker

A personal finance web application for tracking income and expenses. Users can record transactions, organize them by category, filter and paginate through history, and view balance summaries and spending reports broken down by category.

## Core Entities

- **Transaction** — a financial event with a type (`income` or `expense`), amount, date, category, and optional description
- **Category** — a user-defined label for grouping transactions (e.g. "Food", "Transport")

## Key Features

- Create, update, and delete transactions
- Filter transactions by date range, category, and type
- Paginated transaction list
- Balance widget (total income, total expenses, net balance)
- Reports with per-category breakdowns
- Category management (create, rename, delete)
